# GitHub Pages Skills Project

Este repositorio forma parte del aprendizaje práctico sobre cómo utilizar GitHub Pages para crear y publicar sitios web de manera sencilla y gratuita.

En este proyecto, se exploran los conceptos básicos necesarios para transformar un repositorio en una página web accesible públicamente. A lo largo del proceso, se aprende a trabajar con archivos Markdown (`.md`), gestionar contenido y aprovechar las herramientas que ofrece GitHub.

El objetivo principal es desarrollar una comprensión sólida de cómo estructurar contenido, realizar cambios y visualizarlos en línea en tiempo real. Este repositorio sirve como punto de partida para crear sitios personales, documentación o proyectos web simples.
